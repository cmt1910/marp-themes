# fukui

高専プロコンで作成したスライドをベースに作成したテーマです。

| 種類               | 例                                             |
| ------------------ | ---------------------------------------------- |
| タイトルスライド   | ![タイトルスライド](./img/fukui_example_1.png) |
| 通常スライド       | ![通常スライド](./img/fukui_example_2.png)     |
| 画像スライド       | ![通常スライド](./img/fukui_example_3.png)     |
| 画像スライド(分割) | ![通常スライド](./img/fukui_example_4.png)     |

## タイトルスライド

以下のように記述することでタイトルスライドを作成することができます。

```text
<!--
_class: lead
-->

# Fukui theme

## by ちょこみんと

高専プロコンで作成したスライドのデザインをベースに作成したテーマです。
```
